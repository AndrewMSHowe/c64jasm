# C64jasm

## Install c64jasm

Command line assembler:

```
npm install -g c64jasm
```

VSCode extension: Search for `c64jasm` in the VSCode marketplace and install.

## Release history

c64jasm 0.2.0:
- Support "server" mode for debug info.  Required for VSCode+VICE source level debugging.
